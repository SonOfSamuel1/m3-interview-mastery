# Documentation

Planning documents, research materials, reviews, and worksheets for the **Active Offer** course development.

## Directory Structure

```
Documentation/
├── Planning/         # Module planning and enhancement summaries
├── Research/         # External research and reference materials
├── Reviews/          # Course architect feedback and evaluations
└── Worksheets/       # Offer development and framework worksheets
```

## Planning

**Location**: `Planning/`

Contains project planning and module enhancement documentation:
- **M1-ENHANCEMENT-SUMMARY.md**: M1 Profile Matching enhancement plan
- **MINDSET_MODULE_EXPANSION_SUMMARY.md**: M0 Mindset expansion project summary
- **SLIDE-SPLIT-IMPLEMENTATION-PLAN.md**: Multi-slide content restructuring plan

These documents track the evolution and strategic decisions behind module improvements.

## Research

**Location**: `Research/`

External research and reference materials:
- **Perplexity Research on Expert Secrets Book.md**: AI-generated research on Russell Brunson's framework
- **Expert Secrets - Action Checklist.md**: Original action items from Expert Secrets book
- **Expert Secrets - Complete Action Checklist (TB Edits).md**: Customized checklist with Active Offer context

Use these materials to understand the foundational frameworks (Three Secrets, Value Ladder, Perfect Webinar) applied in course design.

## Reviews

**Location**: `Reviews/`

Professional course architect feedback:
- **Course-Review-Feedback.md**: Comprehensive course evaluation and improvement recommendations

This feedback shapes ongoing course iterations and quality improvements.

## Worksheets

**Location**: `Worksheets/`

Offer development and value creation frameworks:
- **Active-Offer-Grand-Slam-Offer-Worksheet.md**: Active Offer-specific offer design worksheet
