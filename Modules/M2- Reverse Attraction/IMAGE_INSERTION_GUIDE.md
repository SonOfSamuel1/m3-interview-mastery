## Slide 8: LinkedIn Sales Navigator Dashboard

### What's Needed
**A screenshot of the LinkedIn Sales Navigator dashboard showing key features and navigation**

### Recommended Content
- Full Sales Navigator dashboard view
- Show search interface and filters
- Include saved searches panel
- Display lead recommendations section
- Capture navigation menu

### Placeholder Specifications
- **Location:** Full width below title
- **Dimensions:** 1680 x 600 pixels (approximately)
- **Position:** Centered, 200px from top

### Step-by-Step Instructions

**Method 1: Using Photoshop/Figma**
1. Open `8.png` in Photoshop or Figma
2. Create a new layer
3. Use Rectangle Select Tool to select the placeholder area (inside gold border)
4. Delete or hide the placeholder layer
5. Import your LinkedIn Sales Navigator screenshot
6. Resize to fit: Width 1680px, Height 600px
7. Position at X: 120, Y: 320
8. Ensure screenshot doesn't overlap title or instruction text
9. Export as PNG (1920x1080, 72 DPI)

**Method 2: Using Canva**
1. Upload `8.png` to Canva as background
2. Add your screenshot as new image layer
3. Resize to cover the gold-bordered placeholder area
4. Position precisely to match placeholder bounds
5. Delete or hide background layer (placeholder)
6. Download as PNG (1920x1080)

**Method 3: Using PowerPoint/Keynote**
1. Set slide size to 1920x1080 (16:9)
2. Insert `8.png` as background
3. Insert your screenshot
4. Resize to match placeholder area
5. Right-click screenshot → Send to Front
6. Export slide as PNG (highest quality)

### Tips
- Take screenshot while logged into Sales Navigator
- Use incognito/private browsing to avoid personal data
- Capture during free trial if you don't have subscription
- Ensure screenshot shows professional interface (clean, organized)
- Hide any personal profile information

## General Guidelines for All Image Insertions

### Image Quality Standards
- **Resolution:** Minimum 72 DPI (150 DPI preferred for screenshots)
- **Format:** PNG for best quality (avoid JPEG compression)
- **Color:** RGB color mode (not CMYK)
- **Size:** Keep file sizes under 2MB per slide

### Visual Consistency
- Match the light gray background (#F5F5F5) where needed
- Ensure images don't have white backgrounds (use transparency or match slide background)
- Keep professional aesthetic aligned with Enhanced-WithGraphics design
- Avoid overly busy or cluttered screenshots

### Privacy & Security
- Remove or blur any personal information
- Don't include real hiring manager names unless you have permission
- Obscure email addresses and phone numbers
- Use sample/anonymized data when possible

### Tools Recommended

**Free Options:**
- **Canva** (canva.com) - Easy drag-and-drop
- **GIMP** (gimp.org) - Free Photoshop alternative
- **PowerPoint/Keynote** - Built-in slide editors
- **Preview (Mac)** - Basic image editing

**Paid Options:**
- **Adobe Photoshop** - Professional editing
- **Figma** - Modern design tool
- **Sketch** - Mac-specific design app
- **Affinity Photo** - One-time purchase Photoshop alternative

**Screenshot Tools:**
- **Snagit** - Advanced screenshot capture
- **CleanShot X (Mac)** - Screenshot with annotation
- **ShareX (Windows)** - Free screenshot utility
- **Built-in tools** - macOS Screenshot (Cmd+Shift+4), Windows Snipping Tool

**Phone Mockup Tools:**
- **Mockuphone.com** - Free online mockups
- **Smartmockups.com** - Professional mockups
- **Placeit.net** - Mockup generator
- **Figma plugins** - Device mockup templates

## Troubleshooting

**Problem: Image doesn't fit placeholder area**
- Solution: Use exact dimensions specified in guide
- Check X/Y coordinates for proper positioning
- Verify you're working at 1920x1080 canvas size

**Problem: Screenshot quality is poor**
- Solution: Take screenshot at higher resolution
- Use retina/HiDPI display if available
- Zoom LinkedIn interface before screenshot
- Export at 150 DPI instead of 72 DPI

**Problem: Colors don't match slide design**
- Solution: Check background is #F5F5F5 (245, 245, 245)
- Adjust screenshot brightness/contrast
- Add subtle shadow for depth
- Ensure PNG has transparency if needed

**Problem: Can't remove placeholder pattern**
- Solution: Select placeholder layer specifically
- Use "Delete Layer" not just eraser
- Ensure you're editing correct layer
- Start with fresh copy of original PNG
