## How to Use

### For Live Webinar Delivery
1. Use `Webinar-v4.2-FINAL-Optimized.pdf` in presentation mode
2. Follow delivery notes in source markdown
3. Combo slides are presentation-ready (custom graphics optional)

### To Regenerate Slides
```bash
cd "Webinar-v4.2-Slides-Optimized"
source ../.venv/bin/activate
python3 generate_slides_v4_2.py
```

### To Update Content
1. Edit `Webinar-Script-v4.2-Optimized.md`
2. Regenerate slides using command above
3. Recompile PDF if needed
