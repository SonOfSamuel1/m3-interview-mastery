## Quick Start

### For Live Webinar Delivery
1. Import slides 1-120 into PowerPoint, Keynote, or Google Slides
2. Review PRESENTATION_GUIDE.md for delivery notes
3. Set up Calendly booking link
4. Replace "BOOK YOUR STRATEGY SESSION" with your actual link on slides 118-120
5. Test screen sharing and audio

### For Replay/Recording
1. Use presentation software's recording feature
2. Or record via Zoom/screen recording software
3. Add chapter markers at each phase for better engagement

### To Customize Slides
1. Navigate to `.Webinar Script v.2/` directory
2. Activate virtual environment: `source venv/bin/activate`
3. Edit `generate_slides.py` (modify content or styling)
4. Run: `python generate_slides.py`
5. Slides regenerate in same directory
